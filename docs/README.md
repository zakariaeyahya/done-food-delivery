# Dossier docs/

Ce dossier contient toute la documentation du projet DONE Food Delivery on Blockchain.

**Rôle** : Guide utilisateur complet pour les clients.

**Contenu** :
- Comment créer un compte
- Comment connecter son wallet MetaMask
- Comment passer une commande
- Comment suivre une livraison
- Comment utiliser les tokens DONE
- Comment ouvrir un litige
- FAQ clients

**Rôle** : Guide complet pour les restaurants.

**Contenu** :
- Comment s'inscrire comme restaurant
- Comment gérer son menu
- Comment traiter les commandes
- Comment confirmer la préparation
- Comment consulter les statistiques
- Comment gérer les revenus
- FAQ restaurants

**Contenu** :
- Comment s'inscrire comme livreur
- Comment effectuer le staking (0.1 ETH)
- Comment accepter des commandes
- Comment suivre les trajets
- Comment confirmer les livraisons
- Comment consulter les gains
- FAQ livreurs

### ADMIN_GUIDE.md

## Utilisation

Ces guides sont destinés à :
- **Utilisateurs finaux** : USER_GUIDE, RESTAURANT_GUIDE, DELIVERER_GUIDE
- **Développeurs** : API_DOCUMENTATION, SMART_CONTRACTS
- **Administrateurs** : ADMIN_GUIDE, TROUBLESHOOTING

Tous les guides sont au format Markdown (.md) pour faciliter la lecture et la maintenance.
